## Key Differences

| Feature | NORTO | ENTHIUM |
|---------|-------|---------|
| **Vowels** | O I A on home row left | C I A E on home row left |
| **Danish Å** | Bottom row (Shift mod) | Home row (no mod) |
| **Danish Ø** | Top left corner | Bottom row (Shift mod) |
| **Danish Æ** | Top row | Bottom row (Ctrl mod) |
| **Thumb letter** | E | R |

## Modifier Legend

| Symbol | Modifier |
|--------|----------|
| GUI | Windows/Command key |
| ALT | Alt/Option key |
| SFT | Shift key |
| CTL | Control key |
| NAV | Navigation layer (hold) |
| MOUSE | Mouse layer (hold) |
