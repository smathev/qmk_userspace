## NORTO (Danish Optimized - Default Layer)

```
╭───────────────────────────────────╮   ╭───────────────────────────────────╮
│  Ø  │  Æ  │  U  │  G  │  J  │     │   │     │  B  │  F  │  L  │  H  │  X  │
├─────┼─────┼─────┼─────┼─────┤     │   │     ├─────┼─────┼─────┼─────┼─────┤
│  O  │  I  │  A  │  T  │  M  │     │   │     │  P  │  N  │  R  │  S  │  D  │
├─────┼─────┼─────┼─────┼─────┤     │   │     ├─────┼─────┼─────┼─────┼─────┤
│ GUI │ ALT │ SFT │ CTL │  C  │     │   │     │  W  │ CTL │ SFT │ ALT │ GUI │
│  ,  │  Y  │  Å  │  V  │     │     │   │     │     │  K  │  Z  │  Q  │  .  │
╰─────┴─────┴─────┼─────┼─────┼─────┤   ├─────┼─────┼─────┴─────┴─────┴─────╯
                  │     │ SYM │ NAV │   │ SPC │BSPC │
                  │     │  -  │  E  │   │     │     │
                  ╰─────┴─────┴─────╯   ╰─────┴─────╯
```

### NORTO Combos (Physical QWERTY positions)
- **Q + W** (Ø + Æ) → ESC
- **E + R** (U + G) → F5 (Reload)
- **R + T** (G + J) → Ctrl+T (New Tab)
- **A + S** (O + I) → TAB
- **O + I** (L + H) → Toggle Mouse Layer
- **O + P** (L + X) → DELETE
- **L + Å** (S + D) → ENTER
- **T + G + B** (T + M + P) → RESET (Left)
- **Y + H + N** (B + P + N) → RESET (Right)

**Right Hand:** Numpad

### Navigation Layer Combos (Nav-specific keycodes)
- **T + PgDn** → Ctrl+W (Close Window)
- **UP + T + PgDn** → Ctrl+Shift+T (Reopen Tab)

## SYMBOLS + FUNCTION KEYS Layer

**Left Hand:** F1-F12  
**Right Hand:** Symbols & XCASE

```
╭───────────────────────────────────╮   ╭───────────────────────────────────╮
│ F1  │ F2  │ F3  │ F4  │ F5  │     │   │     │KEBAB│  (  │  '  │ CAPS│     │
├─────┼─────┼─────┼─────┼─────┤     │   │     ├─────┼─────┼─────┼─────┼─────┤
│ F6  │ F7  │ F8  │ F9  │ F10 │     │   │     │SNAKE│  [  │  @  │  $  │  ~  │
├─────┼─────┼─────┼─────┼─────┤     │   │     ├─────┼─────┼─────┼─────┼─────┤
│ GUI │ ALT │ SFT │ CTL │     │     │   │     │CAMEL│  {  │  <  │  ^  │  `  │
│ F11 │ F12 │     │     │     │     │   │     │     │     │     │     │     │
╰─────┴─────┴─────┼─────┼─────┼─────┤   ├─────┼─────┼─────┴─────┴─────┴─────╯
                  │     │SETUP│     │   │     │     │
                  ╰─────┴─────┴─────╯   ╰─────┴─────╯
```

**Symbol Notes:**
- KEBAB: Enable kebab-case (dash-separated)
- SNAKE: Enable snake_case (underscore-separated)
- CAMEL: Enable camelCase
- CAPS: Caps Word toggle

## MOUSE NAVIGATION Layer

**Left Hand:** Orbital Mouse movement  
**Right Hand:** Layer switching & mouse buttons

```
╭───────────────────────────────────╮   ╭───────────────────────────────────╮
│     │WhlUp│ UP  │WhlDn│     │     │   │     │NORTO│ENTH │     │     │     │
├─────┼─────┼─────┼─────┼─────┤     │   │     ├─────┼─────┼─────┼─────┼─────┤
│WhlLt│LEFT │DOWN │RIGHT│WhlRt│     │   │     │     │SLOW │FAST │     │     │
├─────┼─────┼─────┼─────┼─────┤     │   │     ├─────┼─────┼─────┼─────┼─────┤
│CrsLt│CrsUp│CrsDn│CrsRt│     │     │   │     │ GUI │ ALT │ SFT │ CTL │     │
╰─────┴─────┴─────┼─────┼─────┼─────┤   ├─────┼─────┼─────┴─────┴─────┴─────╯
                  │     │     │     │   │ BTN1│ BTN2│
                  ╰─────┴─────┴─────╯   ╰─────┴─────╯
```

**Mouse Legend:**
- OM = Orbital Mouse (circular movement)
- Whl = Scroll wheel
- Crs = Cursor (accelerated movement)
- NORTO/ENTH = Exit mouse layer and switch base layer
- SLOW/FAST = Adjust mouse speed
- BTN1/BTN2 = Left/Right click

**Configuration & Debugging**

```
╭───────────────────────────────────╮   ╭───────────────────────────────────╮
│  Q  │  W  │  E  │  R  │  T  │     │   │     │AS_UP│DT_UP│  C  │  V  │  G  │
```

**Setup Keys:**
- AS_UP/DN: Adjust Auto-Shift timing
- AS_RPT: Report Auto-Shift settings
- DT_UP/DN: Adjust Dynamic Tapping Term
- DT_PRT: Print Tapping Term settings

## Layer Access

- **Base → Navigation**: Hold E (NORTO) or R (ENTHIUM)
- **Base → Symbols**: Hold - (minus)
- **Base → Mouse**: Tap combo O+I (QWERTY positions)
- **Any → Setup**: Hold SETUP thumb key in Symbols layer
